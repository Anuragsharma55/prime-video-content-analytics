
#  Prime Video Content Analytics Dashboard (Power BI)

##  Project Overview

Project Overview

This project is a Power BI dashboard to analyze Prime Video’s content library, performance, and profitability.
It provides insights into:

Content distribution (Movies vs TV Shows)

Genre popularity

Ratings categories

Content growth over time

Profit by content type

The goal is to understand how content is structured and how different factors relate to profit and growth.
##  Business Problem

OTT platforms like Prime Video need to understand:

* Which content types generate the highest profit?
* What genres dominate the platform?
* How content production has evolved over time?
* What ratings categories have the most titles?
* How balanced is the catalog between movies and TV shows?

This dashboard answers these questions in a single, interactive executive view.

---

##  Key KPIs

* **Total Profit**
* **Total Titles**
* **Total Genres**
* **Total Ratings Categories**
* **Total Directors**
* **Movies vs TV Shows Count**
* **Profit by Content Type**

---

##  Dashboard Insights

Some examples of insights that can be derived:

* Movies generate significantly higher profit compared to TV Shows.
* Drama and Action dominate the content library.
* Content production has increased rapidly after 2000.
* The platform focuses heavily on 13+ and 16+ rated content.
* Movies form the majority of the catalog.
